DatepickerTabs

A versatile date picker with day and month selection modes

View on GitHub

Date Mode

No date selected

Example Code:

const datePicker = new DatepickerTabs('#date-input', {
  mode: 'day',
  multiple: false,
  dateFormat: 'DD MMM YYYY'
});

Month Mode

No month selected

Example Code:

const monthPicker = new DatepickerTabs('#month-input', {
  mode: 'month',
  multiple: true,
  maxMonthSelection: 3
});
