DatepickerTabs Examples

Basic Usage

// Basic initialization
const picker = new DatepickerTabs('#date-input');

Month Selection with Multiple Dates

// Month picker with multiple selection
const picker = new DatepickerTabs('#date-input', {
    displayType: 'month',        // Only show month picker, no tabs
    maxMonthSelection: 6,        // Limit to 6 months max
    maxDate: new Date(2025, 11, 31), // No selections past Dec 31, 2025
    multiple: true
});

Using Methods

// Setting values programmatically
picker.setDisplayType('tabs');      // Switch to tabbed view
picker.setMaxMonthSelection(3);     // Reduce max selection to 3 months
picker.setMaxDate(new Date(2026, 5, 30)); // Update max date to June 30, 2026
picker.setMultiple(true);            // Enable multiple selection
picker.setDate(new Date());          // Set date to today
